Dear Bash,

I've seen you around quite often
but never on my own screen.
When I got you for my 64-bit
I started to see how much you would mean.
You give me freedom to navigate
all I need to do is use "cd"
and with commands like "nano" and "touch"
I can edit and create files with ease.
You even let me add my own contents
with the command "mkdir"
and after listing my directory's files
"ls" made me realize we'd go far.
I don't worry about losing my location
with "pwd" I'll see right where we are.
"ssh" even takes me to the server
Bash, together we can reach the stars.

Yours Truly,
Eli
